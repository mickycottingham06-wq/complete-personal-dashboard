# DESIGN_DECISIONS.md

## Purpose

Explain WHY important design choices exist.

Claude should understand these decisions rather than replacing them.

---

# Dark Theme

Chosen because:

Less eye strain.

Premium appearance.

Better contrast.

Fits long usage sessions.

---

# Glass Cards

Chosen because:

Modern appearance.

Visual hierarchy.

Premium feel.

Clear separation of information.

---

# Bottom Navigation

Chosen because:

Fast navigation.

Mobile friendly.

Consistent experience.

---

# Cards Instead of Tables

Cards are easier to scan.

Better on mobile.

Cleaner layout.

More modern.

---

# Minimal Text

The dashboard should be understood within 30 seconds.

Information should not require reading paragraphs.

---

# Progress Bars

Humans understand progress visually.

Use progress bars wherever appropriate.

---

# AI Assistant

Designed to reduce thinking.

Not increase it.

---

# Dashboard First

The dashboard should always provide answers.

Not simply display information.

---

# Final Rule

Every design decision should improve usability before improving appearance.
