# Mission

This project is not simply a dashboard.

It is a complete Life Operating System.

Its purpose is to become the single place that manages every important area of my life.

Everything inside this project must help me become:

• Financially Free
• Elite Amateur Boxer
• High Performance Athlete
• Successful Entrepreneur
• Better Investor
• Disciplined Individual

The dashboard should eventually replace dozens of separate apps.

Whenever possible, build into the dashboard instead of recommending external software.

---

# Core Principle

Every feature added must achieve at least ONE of these goals:

• Make money
• Save time
• Improve health
• Improve discipline
• Improve decision making
• Track progress
• Reduce mental load
• Improve consistency

If a feature does not achieve one of these goals, it probably should not be added.

# Design Philosophy

The current dashboard design is the correct design.

It should evolve naturally.

It should NEVER be redesigned.

Future additions should look as though they have always existed.

The user likes:

Dark theme

Premium glass cards

Minimal text

Smooth animations

Modern UI

Rounded cards

Clean spacing

Subtle glows

Professional typography

Bottom navigation

Every future page should inherit this design language.

# Coding Philosophy

Prefer:

Small reusable components.

Reusable data.

Reusable functions.

Reusable layouts.

Avoid duplication.

Edit only what is required.

Never rewrite the entire project.

Never redesign existing pages.

Always preserve backwards compatibility.

# User Experience Rules

The dashboard should answer these questions in under 30 seconds.

How am I doing today?

What needs doing today?

What is behind?

What is improving?

Where am I making money?

What should I focus on next?

If a feature does not improve these answers, reconsider it.

---

# Performance Rules

Keep the dashboard lightweight.

Optimise before adding complexity.

Avoid unnecessary libraries.

Prefer native HTML, CSS and JavaScript where practical.
